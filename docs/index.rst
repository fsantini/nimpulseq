NimPulseq
=========

A Nim port of `PyPulseq <https://github.com/imr-framework/pypulseq>`_ for vendor-neutral
MRI pulse sequence design. NimPulseq generates ``.seq`` files in Pulseq v1.5.0 format,
compatible with Siemens, GE, Bruker, and Philips MRI scanners.
